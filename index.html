ในคอร์สนี้คุณจะได้ลงมือทำอะไรบ้าง

PART 1 — CORE 3 ชม.

เอา idea ขึ้น live ด้วย Vercel + Supabase

จบพาร์ทนี้: คุณมีเว็บ/แอปที่เปิดจากมือถือใครก็ได้ แล้วใช้งานได้จริง

  • ▸ เข้าใจภาพรวมระบบแบบภาษาคน: หน้าร้าน / หลังร้าน / คลังของ
  • ▸ เอาหน้าร้านขึ้น live ด้วย Vercel — ได้ URL จริง + HTTPS อัตโนมัติ
  • ▸ ต่อฐานข้อมูล + ระบบสมาชิกด้วย Supabase (มี template ให้)
  • ▸ เชื่อมทุกอย่างเข้าด้วยกัน + ตั้งค่าความปลอดภัยไม่ให้กุญแจหลุด
  • ▸ ใส่ชื่อเว็บ (domain) ของคุณเอง ให้ดูเป็นมืออาชีพ
PART 2 — ADVANCED 3 ชม. โบนัส

เมื่อแอปโตเกิน Vercel → เซิร์ฟเวอร์ของคุณเอง

สำหรับคนที่ทดสอบตลาดผ่านแล้ว และอยากคุมระบบ + scale ได้เอง

  • ▸ Docker: แพ็กระบบให้รันที่ไหนก็ได้ (มี template ให้)
  • ▸ AWS EC2: เช่าเซิร์ฟเวอร์ของตัวเอง + รับเครดิตฟรี $100
  • ▸ Deploy ขึ้นเซิร์ฟเวอร์จริง + ต่อ domain + HTTPS เต็มรูปแบบ

Part 2 คือของแถมมูลค่าสูงที่ไม่บีบให้คุณทำตอนยังไม่พร้อม — เริ่มจาก Core ให้แอป live ก่อน แล้วค่อยมาต่อเมื่อพร้อม scale
